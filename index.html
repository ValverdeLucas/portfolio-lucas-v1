  • Challenge ONE: Forum HUB API

    Jul/2024

    O Fórum HUB API é um desafio de conclusão do curso de backend (Java e Spring) do Oracle Next Education e consiste em uma API Rest com CRUD para criação, remoção, atualização e listagem de posts por usuários, além de persistência dos dados com a JPA em um banco de dados relacional. Também foi aplicada uma camada de segurança que apenas permite que usuários com um token de autorização possam fazer qualquer requisição a API.

    Ferramentas utilizadas: Spring Boot, JPA, Hibernate, MySQL e Spring Security

    Github:

    https://github.com/ValverdeLucas/Challenge-ONE-ForumHUB



  • Challenge ONE: Decodificador de Texto

    Fev/2024

    O decodificador de texto é uma aplicação toda feita com HTML + CSS e JavaScript que codifica/decodifica o texto com as seguintes regras: a = ai / e = enter / i = imes / o = ober / u = ufat. Também possui um “dark mode” para reversão das cores da página e responsividade de tela para uso em desktops, tablets e mobiles.

    Ferramentas utilizadas: HTML, CSS e JavaScript.

    Github:

    https://github.com/ValverdeLucas/projeto-one-decodificador



  • Labecommerce (React)

    Nov/2023

    O Labecommerce trata-se de uma loja toda criada em React com filtros para visualização dos produtos em ordem alfabética e preço. Sendo possível incluir itens ao carrinho com atualização dos valores e quantidades em tempo real, uma galeria dinâmica e recuperação dos dados da sessão através do salvamento do estado dos itens no local storage.

    Principais disciplinas: HTML, CSS e JavaScript, React.

    Github:

    https://github.com/ValverdeLucas/projeto-frontendreact
